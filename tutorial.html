Game Tutorial

Wat ga je hier allemaal leren?

  1. Hoe maak je een website met HTML?
  2. Hoe style je een website met CSS?
  3. Hoe maak je een interactieve website met JavaScript?

Wat is HTML? HTML staat voor HyperText Markup Language. Dit word gebruikt om de structuur en inhoud van een webpagina te definiëren. Dit geeft alles aan via tags. Deze tags hebben allemaal een eigen functie. Zoals de tag <h1> voor de hoofdtitel. Zo een tag open en sluit je altijd door <h1> en </h1> te gebruiken. Tussen deze tags zet je dan de tekst die je wilt weergeven. Zoals in dit geval "Game Tutorial". Er zijn veel verschillende tags die je kunt gebruiken, zoals <div> voor het maken van een container, <p> voor het maken van een paragraaf, <a> voor het maken van een link, en nog veel meer. HTML is de basis van elke website en is essentieel om te leren als je een webontwikkelaar wilt worden.

Wat is CSS? CSS staat voor Cascading Style Sheets. Dit word gebruikt om de stijl en lay-out van een webpagina te definiëren. Met CSS kun je de kleuren, lettertypen, marges, en nog veel meer van een webpagina aanpassen. CSS werkt door middel van selectors, waarmee je specifieke HTML-elementen kunt selecteren en stylen. Je kunt CSS in een apart bestand plaatsen of direct in de HTML-code schrijven. CSS is essentieel om een aantrekkelijke en gebruiksvriendelijke website te maken.

Wat is JavaScript? JavaScript is een programmeertaal die wordt gebruikt om interactieve elementen aan een webpagina toe te voegen. Met JavaScript kun je dingen doen zoals het reageren op gebruikersinvoer, het manipuleren van de inhoud van een webpagina, en nog veel meer. JavaScript werkt door middel van functies, waarmee je specifieke taken kunt uitvoeren. Je kunt JavaScript in een apart bestand plaatsen of direct in de HTML-code schrijven. JavaScript is essentieel om een dynamische en interactieve website te maken.

De interactie met de velden

De 3 Event Listeners Uitgelegd

Denk aan event listeners als "wachters" die luisteren naar bepaalde dingen die gebeuren, en daar dan op reageren.


1. keydown listener - Luistert wanneer je toetsen indrukt

            inputs.forEach((input, i) => {
                input.addEventListener('keydown', e => {
                    if (input.disabled) return;
                    
                    if (e.key === 'Backspace') {
                        if (input.value) { input.value = ''; }
                        else if (i > 0) { inputs[i - 1].focus(); inputs[i - 1].value = ''; }
                        e.preventDefault();
                    }
                    if (e.key === 'ArrowLeft' && i > 0) inputs[i - 1].focus();
                    if (e.key === 'ArrowRight' && i < inputs.length - 1) inputs[i + 1].focus();
                    });
                });

Deze code voeg je gewoon onder de eerdere const input toe

Wat het doet deze code?

  • 'Backspace': Verwijdert de huidige letter. Als het vakje leeg is, gaat het terug naar het vorige vakje en verwijdert die letter.
  • 'ArrowLeft': Gaat naar het vorige vakje (zoals naar achteren gaan).
  • 'ArrowRight': Gaat naar het volgende vakje (zoals naar voren gaan).

Real-world voorbeeld: Alsof een leraar kijkt welke toetsen je indrukt en daar op reageert.


2. input listener - Luistert wanneer je een letter typt Wat het doet:
  • Staat alleen letters toe (geen nummers of symbolen).
  • Gaat automatisch naar het volgende vakje als je een letter typt.
  • Controleert of de rij compleet is (alle 6 vakjes ingevuld).
  • Als de rij compleet is, controleert het of je gok correct is en gaat naar de volgende rij vakjes.

Real-world voorbeeld: Alsof een helper je werk controleert zodra je elke regel afmaakt.

maar voor we dit doen moeten we eerst 2 funties schrijven, dit zijn stuckjes code die we kunnen hergebruiken. (we zullen deze functies ook nog uitbrijden voor meer funtionaliteit)

Dit stukje code plak je weer gewoon in je JavaScript-bestand onder de eerdere code:


            const lettersPerRow = 6;

            function isRowComplete(startIndex) {
                const endIndex = startIndex + lettersPerRow;
                const rowInputs = inputs.slice(startIndex, endIndex);
                return rowInputs.every(input => input.value.trim() !== '');
            }

            function checkRow(startIndex) {
                const endIndex = startIndex + lettersPerRow;
                const rowInputs = inputs.slice(startIndex, endIndex);

                // STEP 1: Disable and fade row
                rowInputs.forEach(input => {
                    input.disabled = true;
                    input.style.opacity = '0.5';
                });
            }

Wat doet deze code nu?

  • const lettersPerRow = 6; Dit is een simpele instelling. We vertellen het spel dat elk woord in onze game precies 6 letters lang is.
  • Functie isRowComplete (Is de rij vol?): Deze functie is een controleur. Hij pakt het eerste vakje van de huidige rij, telt er 6 af, en kijkt dan of ze allemaal zijn ingevuld. Zodra er ook maar één vakje leeg is, zegt hij "Nee". Zijn ze allemaal vol? Dan geeft hij groen licht.
  • Functie checkRow (De rij blokkeren): Dit is het begin van de functie die straks gaat checken of je het woord goed hebt geraden (met de groene en oranje kleuren). In deze eerste stap doet hij nog maar één ding: het beveiligen van je gok. Hij pakt de 6 vakjes, schakelt ze uit (disabled = true) en maakt ze een beetje doorzichtig (opacity = '0.5'). Zo kan de speler zijn letters niet meer stiekem veranderen!

nu we de fucties hebben kunnen we de input listener schrijven. Deze zal er als volgt uit zien (deze komt onder de keydown event listener binnen de input.foreach loop): 


            input.addEventListener('input', e => {
                if (input.disabled) return;
                
                input.value = input.value.replace(/[^a-zA-Z]/g, ''); // letters only
                if (input.value && i < inputs.length - 1) inputs[i + 1].focus();
                
                // Check if current row is complete and lock it
                const rowStartIndex = Math.floor(i / lettersPerRow) * lettersPerRow;
                if (isRowComplete(rowStartIndex)) {
                    checkRow(rowStartIndex);
                    // Focus the first input of the next row
                    const nextRowStart = rowStartIndex + lettersPerRow;
                    if (nextRowStart < inputs.length) {
                        inputs[nextRowStart].focus();
                    }
                }
            });

Dus als je deze code correct is geplaats ziet de foreach er zo uit (het geslecteerde deel is de nieuwe code):

Wat doet dit nieuw stukje code nu? Deze code is de motor van het spel en luistert naar alles wat je typt. Het werkt in vier stappen:

  • Alleen letters toegestaan (replace): Het vreemde stukje code /[^a-zA-Z]/g is een zogenaamde 'RegEx' (reguliere expressie). Het is een strikte beveiliger: als een speler per ongeluk een cijfer (bijv. 8) of leesteken (bijv. @) typt, wordt dit onmiddellijk gewist en vervangen door helemaal niks (''). Je kunt dus uitsluitend letters typen!
  • Automatisch doorschuiven (focus): Zodra je een geldige letter hebt getypt, zorgt het script ervoor dat je niet zelf met je muis naar het volgende vakje hoeft te klikken. Met inputs[i + 1].focus() springt je typ-cursor automatisch naar het volgende beschikbare vakje.
  • Controleren of je klaar bent met je gok: Met een klein beetje wiskunde (Math.floor) berekent het script in welke van de 6 rijen je momenteel aan het typen bent. Vervolgens gebruikt hij de isRowComplete functie die we eerder hebben gemaakt. Heb je 6 letters getypt en is de rij vol? Dan roept hij de checkRow functie aan om je antwoord vast te zetten.
  • Klaarzetten voor de volgende poging: Als je gok is vastgezet, berekent het spel razendsnel waar de volgende rij begint (nextRowStart). Als je nog pogingen over hebt, zet hij je cursor automatisch in het eerste vakje van die nieuwe rij, zodat je in één vloeiende beweging verder kunt spelen.

3. focus listener - Luistert wanneer je op een vakje klikt Wat het doet:
  • Voorkomt dat je op afgeronde rijen klikt (al geraden).
  • Houdt je gericht op de huidige rij waar je in tikt.
  • Als je op een oude rij klikt, brengt het je terug naar de rij waar je aan werkt.
Real-world voorbeeld: Zoals een portier bij een club die je alleen in de rij laat waar je hoort te zijn. 


            input.addEventListener('focus', () => {
                // Check which row this input is in
                const currentRowStart = Math.floor(i / lettersPerRow) * lettersPerRow;
                
                // Find the first non-disabled row
                let firstEnabledRowStart = 0;
                for (let row = 0; row < inputs.length; row += lettersPerRow) {
                    if (!inputs[row].disabled) {
                        firstEnabledRowStart = row;
                        break;
                    }
                }
                
                // If this input is in the first enabled row, allow focus and select
                if (currentRowStart === firstEnabledRowStart) {
                    input.select();
                } else {
                    // Otherwise, focus the first input of the first enabled row
                    inputs[firstEnabledRowStart].focus();
                }
            });

Deze code plak je onder de code van de vorige stap (het geselecteerde gedeelte is de nieuwe code).

Wat doet dit nieuw stukje code nu? Herinner je je het voorbeeld van de portier nog? Dit is de exacte code die ervoor zorgt dat spelers niet vals kunnen spelen door zomaar ergens op het bord te klikken. Het werkt in drie slimme stappen:

  • 1. Waar heb je geklikt? (currentRowStart): Zodra je op een vakje klikt (het vakje krijgt 'focus'), berekent het script door middel van wat wiskunde in welke van de rijen dit specifieke vakje zich bevindt.
  • 2. Waar hoor je te zijn? (firstEnabledRowStart): Vervolgens scant de code met een zogenaamde for-loop het hele speelveld van boven naar beneden. Hij zoekt en stopt (break) bij de allereerste rij die nog niet is uitgeschakeld (disabled). Dit is de huidige, actieve beurt van de speler.
  • 3. De portier grijpt in (If / Else): Nu vergelijkt de code deze twee locaties met elkaar. Klikte je inderdaad in de actieve rij? Dan is alles goed (de if) en wordt je letter netjes geselecteerd (input.select()) zodat je die makkelijk kunt aanpassen.   • Zo ja: markeert als orange (juiste letter, verkeerde plaats) en trekkt 1 van de beschikbaarheid af
  • Dit zorgt dat letters maximaal zoveel keer oranje worden gemarkeerd als ze voorkomen

                // STEP 5: Mark ORANGE letters (wrong position, but in word)
