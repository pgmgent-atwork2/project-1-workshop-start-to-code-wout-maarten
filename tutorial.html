Game Tutorial

Wat ga je hier allemaal leren?

  1. Hoe maak je een website met HTML?
  2. Hoe style je een website met CSS?
  3. Hoe maak je een interactieve website met JavaScript?

Wat is HTML? HTML staat voor HyperText Markup Language. Dit word gebruikt om de structuur en inhoud van een webpagina te definiëren. Dit geeft alles aan via tags. Deze tags hebben allemaal een eigen functie. Zoals de tag <h1> voor de hoofdtitel. Zo een tag open en sluit je altijd door <h1> en </h1> te gebruiken. Tussen deze tags zet je dan de tekst die je wilt weergeven. Zoals in dit geval "Game Tutorial". Er zijn veel verschillende tags die je kunt gebruiken, zoals <div> voor het maken van een container, <p> voor het maken van een paragraaf, <a> voor het maken van een link, en nog veel meer. HTML is de basis van elke website en is essentieel om te leren als je een webontwikkelaar wilt worden.

Wat is CSS? CSS staat voor Cascading Style Sheets. Dit word gebruikt om de stijl en lay-out van een webpagina te definiëren. Met CSS kun je de kleuren, lettertypen, marges, en nog veel meer van een webpagina aanpassen. CSS werkt door middel van selectors, waarmee je specifieke HTML-elementen kunt selecteren en stylen. Je kunt CSS in een apart bestand plaatsen of direct in de HTML-code schrijven. CSS is essentieel om een aantrekkelijke en gebruiksvriendelijke website te maken.

Wat is JavaScript? JavaScript is een programmeertaal die wordt gebruikt om interactieve elementen aan een webpagina toe te voegen. Met JavaScript kun je dingen doen zoals het reageren op gebruikersinvoer, het manipuleren van de inhoud van een webpagina, en nog veel meer. JavaScript werkt door middel van functies, waarmee je specifieke taken kunt uitvoeren. Je kunt JavaScript in een apart bestand plaatsen of direct in de HTML-code schrijven. JavaScript is essentieel om een dynamische en interactieve website te maken.

Wat heb je nodig om deze tutorial te volgen?

  • Een computer (windows, mac, of linux)

  • Een code editor (zoals Visual Studio Code)

Stap 1: Project opzetten

Om dit project te starten, moet je een nieuwe map aanmaken op je computer. Geef deze map een duidelijke naam, zoals "game-tutorial". Vervolgens kun je een nieuw bestand aanmaken met de naam "index.html". Dit is het bestand waarin je je HTML-code zal schrijven.

Hierna gaan we dit project open in onze code editor bv Visual Studio Code. Dit doe je door visual studio code te openen en en te klikken op Open Folder.

Eenmaal daarop geklikt ga je naar de locatie waar je de map van het project hebt aangemaakt en klik je hierop en dan rechtsonder op open

Nu zie je het project in de bestandsverkenner van visual studio code. Hier kunnen we dan het begin HTML bestand aanmaken. Dit doe je door rechts te klikken op de map en "New File" te kiezen.

Dit nieuw bestand geef je dan de naam "index.html"

Waarom index.html? Index.html word standaard gebruikt als startpagina van een website en de .html extensie zorgt ervoor dat de browser weet dat het om HTML-code gaat.

Stap 2: HTML structuur maken

Nu we ons project hebben opgezet, kunnen we beginnen met het schrijven van onze HTML-code. We beginnen met het toevoegen van de basisstructuur van een HTML-document. Dit doe je door de volgende code in je index.html bestand te plakken:

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Tutorial Game</title>
    </head>
    <body>

    </body>
    </html>

Wat doet dit nu allemaal? De <html> tag is de basis van je pagina. In de <head> staan onzichtbare instellingen zoals de <meta> charset voor speciale tekens en de <title> voor je browsertab (de naam dat bovenaan bij een website staat). Alles wat je uiteindelijk écht op het scherm wilt zien, plaats je tussen de <body> tags.

Hoe werkt zo een input veld juist? Je ziet in zo een input veld verschillende waardes staan namelijk: type, inputmode, maxlength en autocomplete. Deze hebben allemaal een eigen functie. Bij Type staat er nu ="text" dit betekend dat dit een veld voor text is (er zijn namelijk ook nog andere types zoals "number" voor nummers, "email" voor email adressen, etc). Bij inputmode staat er ="char" dit betekend dat we hier karakters willen invoeren. Bij maxlength staat er ="1" dit betekend dat er maar 1 karakter in dit veld mag worden ingevoerd. En bij autocomplete staat er ="off" dit betekend dat de browser geen suggesties mag geven op basis van eerder ingevoerde waarden in dit veld.

De interactie met de velden

De 3 Event Listeners Uitgelegd

Denk aan event listeners als "wachters" die luisteren naar bepaalde dingen die gebeuren, en daar dan op reageren.


1. keydown listener - Luistert wanneer je toetsen indrukt
Wat het doet:
  • Backspace-toets: Verwijdert de huidige letter. Als het vakje leeg is, gaat het terug naar het vorige vakje en verwijdert die letter.
  • Linker Pijl-toets: Gaat naar het vorige vakje (zoals naar achteren gaan).
  • Rechter Pijl-toets: Gaat naar het volgende vakje (zoals naar voren gaan).
Real-world voorbeeld: Alsof een leraar kijkt welke toetsen je indrukt en daar op reageert.


2. input listener - Luistert wanneer je een letter typt Wat het doet:
  • Staat alleen letters toe (geen nummers of symbolen).
  • Gaat automatisch naar het volgende vakje als je een letter typt.
  • Controleert of de rij compleet is (alle 6 vakjes ingevuld).
  • Als de rij compleet is, controleert het of je gok correct is en gaat naar de volgende rij vakjes.
Real-world voorbeeld: Alsof een helper je werk controleert zodra je elke regel afmaakt.

3. focus listener - Luistert wanneer je op een vakje klikt Wat het doet:
